# Judo Passaggi di Cintura - Libretto Tecnico Digitale

Benvenuti nel repository del **Libretto Tecnico Digitale di Judo**, un progetto creato per supportare gli studenti (8-14 anni) della scuola **FUJI SAN JUDO SCHIO A.S.D.** nel loro percorso tecnico.

## 🥋 Descrizione del Progetto

Questo libretto è una risorsa interattiva e mobile-responsive che raccoglie tutte le tecniche necessarie per i passaggi di cintura, dalla bianca alla marrone. L'obiettivo è fornire agli atleti uno strumento facile da consultare direttamente sul tatami via smartphone o tablet.

## ✨ Caratteristiche Principali

- **Organizzazione per Cinture**: Le tecniche sono suddivise chiaramente in base al grado (es. Da Bianca a Mezza Gialla, Da Gialla a Mezza Arancio, ecc.).
- **Video Tutorial**: Ogni tecnica include un link diretto ai video ufficiali del **Kodokan** su YouTube per uno studio visivo accurato.
- **Interfaccia Moderna**: Design pulito, reattivo e con navigazione rapida tramite i "punti cintura" nell'intestazione.
- **Target 8-14 anni**: Linguaggio e struttura pensati per giovani judoisti.

## 🛠️ Tecnologie Utilizzate

- **HTML5**: Struttura semantica.
- **CSS3**: Layout premium con animazioni fluide e design glassmorphism.
- **JavaScript**: Scrolling fluido tra le sezioni.
- **Google Fonts**: Utilizzo dei font *Outfit* e *Inter* per una leggibilità superiore.

## 🚀 Come Visualizzare il Progetto

Il progetto è ospitato su GitHub Pages (o può essere visualizzato localmente aprendo `index.html`).

Se hai attivato GitHub Pages sul repository, sarà visibile all'indirizzo corrispondente (es. `https://petmax1973.github.io/judo_web/`).

## 📂 Struttura del Repository

- `index.html`: Pagina principale del libretto.
- `style.css`: Foglio di stile con il design system completo.
- `booklet_assets/`: Cartella contenente le risorse grafiche (copertina, immagini).
